# kdbackdoor
This project is inspired by the idea that backdoor attack of neural network can be transferred by knowledge distillation.

## Dependency
This project is based on 
* python 3.6.5
* tensorflow-gpu 2.1rc
* numpy 1.18.0
* tqdm 4.43.0

## How to run
```shell
python ./mnist_kdbackdoor.py
```
